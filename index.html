Getting started

Set it up in five steps

This is the actual workflow: download the Marketing folder, give Claude access to it, trigger the design system skill, give Claude the content, and get a polished Clozd-branded deliverable.

Core workflow
1

Download the Marketing folder to your desktop using the ZIP link above, then unzip it so the folder lives on your desktop. It should contain the Clozd slide backgrounds, icons, and logos.

2

Give Claude access to the newly downloaded Marketing folder on your desktop so Claude can reference those brand assets while building your output.

3

Trigger the design system skill using the plugin path, the slash command /applying-clozd-design-system, or by clearly asking for a Clozd-branded output in plain English.

4

Give Claude the content — a topic, notes, transcript, research, rough draft, or existing presentation.

5

Get the branded deliverable — Claude applies Clozd's design system and produces a polished deck, document, one-pager, landing page, or other asset.

Fastest ways to trigger it
  • Plugin path — Open Claude desktop, click Cowork, click the + in the chat box, hover over Plugins, and select applying-clozd-design-system.
  • Slash command — Type /applying-clozd-design-system in the chat box, then give Claude your content or brief.
  • Natural language — Ask Claude to create something and include phrases like make it Clozd-branded or apply Clozd branding.
Try prompts like
Use the Marketing folder on my desktop and make this Clozd-branded Apply Clozd branding to this deck using the desktop Marketing assets Turn this into a Clozd one-pager with the design system

What “Clozd-branded” actually means
The design standards Claude applies

When you request Clozd branding, Claude applies a complete system — not just colors. Every branded output follows these standards:

What gets applied
Colors: Patina (#09D4C8), Blue (#14A6F8), Royal Blue (#274FDB) — gradient order strictly maintained Typography: Inter font family with defined heading hierarchy (48pt, 36pt, 24pt, 16pt) Layout: One idea per slide, generous whitespace, minimal bullets, strong visual hierarchy Assets: Official Clozd backgrounds, icons, and logos sourced from the connected Marketing folder

Getting the most out of Claude
Tips for best results
A few simple habits make your branded outputs sharper, more consistent, and easier to use.
1

Keep the Marketing folder on your desktop

The simplest setup is to keep the unzipped Marketing folder on your desktop so Claude can reliably reference the same Clozd asset package each time.

2

Specify the output type

Claude produces sharper results when you say what you want — a deck, a report, a one-pager, a landing page. Be explicit.

3

Give the skill real source material

Paste notes, upload a draft, share research, or describe the topic clearly. Better content in means a stronger branded deliverable out.

4

Use one conversation from start to finish

Research, write, refine, and brand — all without starting over. Claude retains the thread context, so the final output reflects everything that came before it.
